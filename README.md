# Travel Booking Homepage

A modern, responsive travel booking website homepage that matches the design specifications with a dark theme and beautiful gradient backgrounds.

## Features

- **Modern Dark Theme**: Sleek dark design with gradient backgrounds
- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Interactive Search Form**: Functional search with destination, dates, and travelers
- **Destination Cards**: Clickable cards for popular destinations (Paris, Kyoto, Mahch Piccro)
- **Carousel Indicators**: Animated background carousel with 5 different gradient themes
- **Newsletter Signup**: Functional newsletter subscription form
- **Social Media Links**: Integrated social media icons
- **Smooth Animations**: Hover effects, transitions, and parallax scrolling

## Getting Started

1. Open `index.html` in your web browser
2. The page will load with smooth fade-in animation
3. Try the search form and interactive elements
4. Click on destination cards to auto-populate the search
5. Subscribe to the newsletter in the footer

## Customization

- Modify gradient colors in `styles.css` under the `.hero-background` section
- Add more destinations by updating the destination cards HTML and CSS
- Customize the carousel timing in `script.js` (currently 5 seconds)
- Update social media links in the footer section

Enjoy your beautiful travel booking homepage! 🌍✈️
